# 🚀 Data Pilot

## 📊 Overview
Data Pilot is a desktop-based login and inventory management system built using C# (.NET) and MySQL.

## 🔐 Features
- Secure Login System
- User Management
- Inventory Management
- CRUD Operations (Create, Read, Update, Delete,Search,Export Excel,Export PDF)
- Search Functionality

## 🗂 Inventory Fields
- Product Name
- Description
- Price
- Quantity

## 🛠 Technologies Used
- C# (.NET Windows Forms)
- MySQL Database

## 💻 How to Run
1. Open project in Visual Studio
2. Setup MySQL database
3. Update connection string
4. Run the project
